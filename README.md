# SauceApp

A simple python web app to search for an Anime using an Image

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install -r requirements.txt
```

## Usage

```coffee
python3 base.py
```
